## Contributors

Contributors are automatically added to this list when their pull requests are merged. You can contribute by:

- Submitting bug fixes
- Adding new features
- Improving documentation
- Reporting issues
- Reviewing pull requests
- Helping other users

1. **Code Contributions**
   - Fix bugs
   - Add features
   - Improve performance
   - Add tests

2. **Documentation**
   - Fix typos

   - Improve clarity
   - Translate docs

3. **Community**
   - Answer questions
